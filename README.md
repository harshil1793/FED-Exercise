# FED-Exercise

This project is created with Angular 9.

## Live preview

https://stranger-things-app-68cd9.firebaseapp.com/

## Getting Started

1. Clone this repository

   ```bash
   git clone https://github.com/harshil1793/FED-Exercise.git
   cd FED-Exercise
   ```

1. Install the npm packages

   ```bash
   npm install
   ```

1. Run the app!

   ```bash
   npm start
    ```
